## Cost Optimization Strategies

1. **Lambda Memory Optimization**
   - Right-size memory allocation
   - Current: 256-512MB
   - Monitor CloudWatch metrics

2. **S3 Lifecycle Policies**
   - Archive old images to S3 Glacier
   - Delete thumbnails after X days
   - Intelligent-Tiering for originals

3. **RDS Instance Sizing**
   - Start with small instance (db.t3.micro)
   - Scale based on connection metrics
   - Consider Aurora Serverless v2

4. **API Gateway Caching**
   - Cache GET responses (list, metadata)
   - TTL: 5-60 minutes
   - Reduce Lambda invocations

5. **CloudWatch Logs Retention**
   - Set retention to 7-30 days
   - Use Logs Insights for analysis
   - Export long-term logs to S3
